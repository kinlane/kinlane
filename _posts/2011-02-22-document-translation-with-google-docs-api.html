--- layout: post title: 'Document Translation with Google Docs API' source: http://kinlane.com/2011/02/22/document-translation-with-google-docs-api/ domain: kinlane.com image: http://kinlane-productions2.s3.amazonaws.com/mimeo/document-translation.jpg --- Google Docs Listing API provides document translation at file upload. When upload a document you can enable document translation with the following parameters:
  • targetLanguage=x
  • sourceLanguage=y
Google can also autodetect the source language is one is not provided.

Google Docs List API provides translation in 53 languages and adding more as Google Translate evolves.
